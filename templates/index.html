Sobre mí

Un dev que construye cosas reales.

Tengo 18 años y soy desarrollador Full Stack Junior de Córdoba, Argentina. Desde temprano me apasionó la programación como herramienta para resolver problemas reales y aprender cómo funcionan los sistemas por dentro.

Me especializo en el desarrollo de aplicaciones web utilizando Python y Flask para el backend, y HTML, CSS y Tailwind CSS para el frontend. También trabajo con bases de datos como MySQL, PostgreSQL y MongoDB, desarrollando aplicaciones seguras, escalables y mantenibles.

Además, me interesa el área de la ciberseguridad y el hacking ético. Estudio análisis de vulnerabilidades, pentesting de aplicaciones, seguridad en redes y buenas prácticas de desarrollo seguro. Utilizo herramientas y metodologías orientadas a identificar, comprender y reportar vulnerabilidades de forma responsable, siempre con fines educativos y de mejora de la seguridad.

Completé cursos de Python avanzado, Python para desarrollo web, HTML y CSS, desarrollo de aplicaciones de escritorio con Python, desarrollo de aplicaciones con inteligencia artificial y continúo formándome en ciberseguridad, pruebas de penetración y seguridad ofensiva para ampliar mis conocimientos y aplicarlos en proyectos reales.
